# ChatGPT File Upload Limits

## ⚠️ Important Information

### Free Tier Accounts

| Limit | Value |
|--------|-------|
| **Uploads per day** | ≈ 3 files |
| **Max file size (general)** | 512 MB per file |
| **Max file size (text)** | 2 million tokens |
| **Max file size (spreadsheets)** | 50 MB |
| **Max file size (images)** | 20 MB |
| **Total storage** | 10 GB |

### Impact on Extension

> **⚠️ WARNING**: Free tier users can only clone **3 conversations per day**.

After reaching the daily upload limit of 3 files, ChatGPT will block new uploads until the next day.

### Recommendations

1. **Use sparingly**: Only clone important conversations.
2. **Consider Plus**: ChatGPT Plus subscribers have much higher limits.
3. **File sizes**: Longer conversations generate larger files.
   - ~100 messages ≈ 50-100 KB
   - ~1000 messages ≈ 500 KB - 1 MB

### Error Message

If you reach the limit, ChatGPT will show:
```
"You've reached your daily upload limit. Try again tomorrow."
```
